  - **본문 일제 점검 결과 (2026-05-05)**: docs 전체에서 `사용자` 단독 사용 60+자리를 점검. 압도적 다수가 T3로 명백히 사용되어 분리 등재가 본문 의미와 일치함을 확인. **T2 의미인데 `사용자`로 적힌 2자리를 발견·수정함**:

## 등재 보류·제외 결정 기록

후보로 검토되었으나 일상어/일회성 예시로 판단해 등재하지 않은 용어들. 같은 후보가 다음 추출 사이클에 다시 올라오면 "이미 검토함"이 보이도록 함.

| 후보 | 등재 제외 이유 | 재검토 트리거 |
|------|---------------|---------------|
| 모듈 | 일반 소프트웨어 어휘. 제품 내부 정의 없음. | feature·종단 의존성 안에서 "모듈"이 별도 카테고리/단위로 정의되면 등재. |
| 컴포넌트 | 일반 어휘. UI 컴포넌트, 미들웨어 컴포넌트 등 문맥별로 다른 의미를 가지지만 별도 정의가 없음. | "UI 컴포넌트"·"미들웨어 컴포넌트" 등 분리된 정의가 PRD에 등장하면 등재. |
